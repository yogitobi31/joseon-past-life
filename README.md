# 내가 전생에 조선인이었다니

모바일 중심 한국어 심리테스트 웹서비스입니다. 12개 질문에 답하면 조선시대 전생 캐릭터 12종 중 하나를 결과로 보여줍니다.

## 구성 파일

- `index.html`: 단일 페이지 구조(시작/질문/결과 화면)
- `style.css`: 한지 질감 + 먹색/남색/낙관 포인트 기반 스타일
- `script.js`: 12문항 로직, 점수 계산, 결과 렌더링, 공유 기능

## GitHub Pages 배포 방법

1. GitHub 저장소에 이 파일들을 푸시합니다.
2. 저장소 **Settings → Pages**로 이동합니다.
3. **Build and deployment**에서 Source를 `Deploy from a branch`로 선택합니다.
4. Branch를 `main`(또는 배포 브랜치) / `/ (root)`로 설정 후 저장합니다.
5. 발급된 Pages URL에서 서비스가 바로 실행됩니다.

## 로컬 실행

서버 없이 동작하므로 `index.html`을 브라우저에서 직접 열어도 됩니다.
